Introduction

ASL Tutor is a interactive program that will help you learn, understand, and practice the alphabet for American Sign Language

The Next Letter button will move on to a different letter in the American Sign Language alphabet for you to practice

The Start button will begin testing to see how well you can sign the current letter shown and help you improve the accuracy of it

The Test Yourself button will hide the display that shows what the current letter is in ASL so you can test your memory

Once clicked it will change to a Show Answer button that will make the letter in ASL visible again
